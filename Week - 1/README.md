# Week-1: Area Calculator Program

## Description
This Python project calculates the area of four shapes: Circle, Triangle, Square, and Rectangle.  
All functions are written in the module area_calculations.py.  
The execute.py script takes input from the user for dimensions and displays the calculated areas.

## How to Run
1. Keep area_calculations.py and execute.py in the same folder.  
2. Run the execute script:

```bash
python execute.py
